# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.0.3] - 2026-02-14
### Added
- 更改接收区TX RX INFO样式为高亮块,更名为包类型显示，并在option菜单中添加开关
- 新增复制成功提示功能
- 新增接收区右键菜单并优化行显示效果
- 新增显示接收区数据长度功能
- 新增接收区重复筛选功能
- 添加新数据闪烁动画来解决数据更新不可见的问题
- 新增工作空间相关内容和逻辑

### Fix
- 改善接收区内容间距
- 修复定时发送功能
- 改善窗口缩小时功能按钮挤压问题
- 重构下方状态条,优化颜色和显示内容
- 优化标题栏按钮像素问题
- 优化sessions菜单和软件左上角的显示内容
- 优化session显示格式
- 修复输入框为空时的发送问题
- 优化命令列表中的命令的默认名称,添加命令名称唯一逻辑
- 改善消息提示框样式
- 优化消息提示框动画

### Removed
- 删除侧边栏的用户功能
- 删除侧边栏未开发的git和search功能
