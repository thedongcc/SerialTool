# Changelog

All notable changes to this project will be documented in this file.

## [0.0.3] - 2026-02-14
### Added
- 更改接收区TX RX INFO样式为高亮块,更名为包类型显示，并在option菜单中添加开关
- 新增复制成功提示功能
- 新增接收区右键菜单并优化行显示效果
- 新增显示接收区数据长度功能
- 新增接收区重复筛选功能
- 添加新数据闪烁动画来解决数据更新不可见的问题

### Fix
- 改善接收区内容间距
- 修复定时发送功能
- 删除侧边栏未开发的git和search功能
- 改善窗口缩小时功能按钮挤压问题
