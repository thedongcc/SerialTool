# 项目核心上下文

## 1. 项目概述
本项目旨在开发一个功能强大、界面现代化的跨平台串口调试助手（Serial Debug Assistant）。
主要平台为 Windows。
目标是提供超越传统串口助手的体验，支持极其丰富的扩展性和智能化功能。

## 2. 核心功能需求
- **现代化的 UI**: 界面美观、流畅，支持深色模式等现代特性。
- **多标签页系统**: 支持同时打开多个调试会话。
- **分栏操作**: 支持灵活的界面布局。
- **串口调试**: 基础的串口收发功能。
- **扩展/插件系统**:
    - CAN 助手
    - Modbus 助手
    - 自定义协议解析
- **图形绘制**: 基于串口数据实时绘制波形/图表。
- **自定义组件**: 用户可以作为“画布”拖拽组件。
- **自定义工作流**: 可编排的数据处理流程。
- **AI 辅助**: AI 帮助配置工作流、分析数据。
- **配置文件**: 支持导入导出。
- **云端功能**:
    - 微信登录
    - 云端配置存储
- **技术适应性**: 架构需灵活，能适配未来新技术。

## 3. 技术约束与偏好
- **开发工具**: JetBrains WebStorm (TypeScript 项目)
- **运行平台**: Windows (通过 Electron 等技术实现跨平台)
- **语言**: TypeScript

## 4. 目录结构
- `src/`: 源代码
- `contexts/`: 项目文档
