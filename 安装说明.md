# SerialTool 安装说明

## ⚠️ 重要提示

如果你之前已经运行过安装程序，软件可能已经安装到系统中了。

### 如何确认是否已安装

1. 检查桌面是否有 "SerialTool" 快捷方式
2. 检查开始菜单是否有 SerialTool
3. 检查以下目录是否存在：
   - `C:\Users\<你的用户名>\AppData\Local\Programs\SerialTool`

### 如果已安装，如何卸载

**方法1：使用卸载程序**
1. 打开"设置" → "应用" → "已安装的应用"
2. 搜索 "SerialTool"
3. 点击"卸载"

**方法2：手动卸载**
1. 找到安装目录（通常在 `C:\Users\<用户名>\AppData\Local\Programs\SerialTool`）
2. 运行 `Uninstall SerialTool.exe`

### 重新安装

卸载完成后，重新运行：
`p:\Webstorm\SerialTool_V1\release\0.0.1\SerialTool-Windows-0.0.1-Setup.exe`

这次应该会看到完整的安装向导界面。

---

## 📋 安装向导应该包含的步骤

1. **欢迎页面** - Welcome to SerialTool Setup
2. **许可协议** - License Agreement  
3. **选择安装位置** - Choose Install Location
4. **快捷方式选项** - Choose Components (桌面快捷方式等)
5. **安装进度** - Installing
6. **完成** - Completing SerialTool Setup

如果仍然直接启动软件，说明NSIS配置有问题，我会进一步调整。
